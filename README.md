# Customer Churn Analysis Dashboard

A Streamlit-based dashboard for analyzing customer churn patterns.

## Features

- Customer churn distribution analysis
- Churn breakdown by gender, subscription type, and support calls
- Total spend vs churn correlation
- Interactive filters and visualizations

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

The app will open at `http://localhost:8501`

## Requirements

- streamlit
- pandas
- matplotlib
- seaborn
- plotly
